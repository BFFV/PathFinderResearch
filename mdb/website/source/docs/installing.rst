##########
Installing
##########
We supply packages for some operating systems which you can use to install MillenniumDB.
If we do not have a package for your system, check out :doc:`docker` or :doc:`compiling`.

.. tabs::

   .. tab:: Linux

      We currently supply **.deb** packages for Ubuntu and Debian based distributions.

      .. tabs::

         .. tab:: Ubuntu based

            - :download:`Ubuntu 20.04 <../../../deb-package/output/millenniumdb_0.1-1_ubuntu-20.04_amd64.deb>`
            - :download:`Ubuntu 22.04 <../../../deb-package/output/millenniumdb_0.1-1_ubuntu-22.04_amd64.deb>`

         .. tab:: Debian based

            - :download:`Debian 10 <../../../deb-package/output/millenniumdb_0.1-1_debian-10_amd64.deb>`
            - :download:`Debian 11 <../../../deb-package/output/millenniumdb_0.1-1_debian-11_amd64.deb>`
            - :download:`Debian 12 <../../../deb-package/output/millenniumdb_0.1-1_debian-12_amd64.deb>`

   .. tab:: Windows

      On windows you can use WSL and install the corresponding Linux package, usually Ubuntu 20.04

      .. tabs::

         .. tab:: Ubuntu based

            - :download:`Ubuntu 20.04 <../../../deb-package/output/millenniumdb_0.1-1_ubuntu-20.04_amd64.deb>`
            - :download:`Ubuntu 22.04 <../../../deb-package/output/millenniumdb_0.1-1_ubuntu-22.04_amd64.deb>`

         .. tab:: Debian based

      However, you can :doc:`compile from source <compiling>` or use :doc:`docker`.
