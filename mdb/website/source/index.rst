##########################
MillenniumDB Documentation
##########################
MillenniumDB is a graph oriented database management system developed by the IMFD [#IMFD]_.

The main objective of this project is to create a fully functional and easy-to-extend DBMS that
serves as the basis for testing new techniques and algorithms related to databases and graphs.
We aim to support multiple graph models. RDF/SPARQL support is fairly complete and we are also
working on a variation of property graphs.

.. Tip::

   We recommended reading :doc:`docs/installing` and :doc:`docs/getting-started` first.

.. Attention::

   This project is still in active development and is not production ready yet, some functionality is missing and there may be bugs.


.. [#IMFD] Millennium Institute for Foundational Research on Data, `link <https://imfd.cl/>`_
